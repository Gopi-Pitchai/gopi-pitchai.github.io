Getting Started with Ollama on your Local System

This document explains setting up Ollama to utilize LLMs on localy. Local deployment ensures data privacy, cost efficiency, and customization.

Ollama Installation

    Download the Ollama installer for Windows from https://ollama.com/download/windows. Execute the downloaded file to initiate the installation process.

    By default, Ollama is installed to C:\Users\<USERNAME>\AppData\Local\Programs\Ollama, where <USERNAME> represents the current user's account name.

    For installation in a custom directory, adhere to the following procedure:
    1. In Command Prompt, navigate to the directory containing the Ollama setup file using the command cd /d <OLLAMA SETUP FILE PATH>. For example: cd /d "C:\Users\<USERNAME>\Downloads\".
    2. Execute ollamasetup.exe /DIR="d:/ollama" to install Ollama in the designated D:\ollama directory.

Download Ollama Models

    To download models from Ollama, execute the following command in Command Prompt: ollama pull . For example, ollama pull deepseek-r1.

    By default, model files are downloaded to C:\Users\<USERNAME>\.ollama\models. To specify a custom download directory, configure the environment variable OLLAMA_MODELS to your preferred folder. Follow these steps:

    For example, set OLLAMA_MODELS to a custom path (e.g., D:\CustomModels) as shown below:

    After configuring the variable, executing ollama pull should download models to the designated custom folder. If the models do not download to the new location, restart Ollama, then close and reopen Command Prompt before retrying.

    With the models successfully downloaded, the following sections will explore the various methods for utilizing local large language models (LLMs) obtained through Ollama.

import matplotlib.pyplot as plt
from PIL import Image
from io import BytesIO
import requests
import ollama

# Download the image from a URL
url = "https://static.remove.bg/sample-gallery/graphics/bird-thumbnail.jpg"
response = requests.get(url)
if response.status_code == 200:
    image = Image.open(BytesIO(response.content))
    plt.imshow(image)
    plt.axis('off')
    plt.show()
else:
    print("Failed to download image. Status code:", response.status_code)

# Convert the image to bytes
image_bytes = BytesIO()
image.save(image_bytes, format=image.format)
image_bytes = image_bytes.getvalue()
print("Image converted to bytes successfully!")

# Use Ollama to pass the image to the model
response = ollama.chat(
    model="llama3.2-vision",
    messages=[{
      "role": "user",
      "content": "Describe the Image",
      "images": [image_bytes]
    }],
)
cleaned_text = response['message']['content'].strip()
print(cleaned_text)

from langchain_ollama import ChatOllama
from langchain_core.prompts import ChatPromptTemplate

# Initialize the Ollama language model
llm = ChatOllama(
    model="llama3.2:1b-instruct-fp16",
    temperature=0,
    num_predict=1024  # maximum number of tokens to generate
)

# Method 1 - Direct invocation of the model
llm.invoke("What is electroencephalography?")

# Method 2 - Using a prompt template and chain
template = """{question}"""
prompt = ChatPromptTemplate.from_template(template)
chain = prompt | llm
Ollama_gen_text = chain.invoke({"question": "What is electroencephalography?"})

# Print the generated response
print(Ollama_gen_text.content)

Electroencephalography (EEG) is a non-invasive medical test that measures the electrical activity of the brain. It's a technique used to record and analyze the brain's electrical signals, which are generated by the neurons in the brain.

During an EEG test, electrodes are placed on the scalp to capture the electrical impulses produced by the brain's neural activity. These electrodes are usually small, flat discs or pads that are attached to the skin using a conductive gel or adhesive.

The EEG signal is typically measured in millivolts (mV) and can be recorded over a wide range of frequencies, from low-frequency ripples (0-30 Hz) to high-frequency oscillations (100-500 Hz). The signal is then analyzed using specialized software to identify patterns, rhythms, and other characteristics of the brain's electrical activity.

EEG is commonly used in various fields, including:

1. Neurology: To diagnose and monitor conditions such as epilepsy, seizures, and brain injuries.
2. Psychiatry: To assess cognitive function, mood, and emotional states in patients with psychiatric disorders.
3. Sleep medicine: To study sleep patterns and identify potential sleep disorders.
4. Neurosurgery: To evaluate the electrical activity of the brain during surgical procedures.
5. Research: To investigate the neural basis of various phenomena, such as perception, attention, and memory.

EEG is a valuable tool for understanding the brain's function and behavior, and it has many applications in both clinical and research settings.

from langchain_ollama import OllamaLLM
from langchain_core.prompts import ChatPromptTemplate
import matplotlib.pyplot as plt
from PIL import Image
from io import BytesIO
import base64
import requests

# Initialize the Ollama vision model
image_llm = OllamaLLM(
    model="llava:7b",
    temperature=0,
    num_predict=1024  # maximum number of tokens to generate
)

# Download the image from a URL
image_url = "https://static.remove.bg/sample-gallery/graphics/bird-thumbnail.jpg"
response = requests.get(image_url)

# Convert the image to Base64 and prepare for display
base64_image = base64.b64encode(response.content).decode('utf-8')
print("Image converted to base64 successfully!")
image_data = base64.b64decode(base64_image)
image = Image.open(BytesIO(image_data))

# Display the image using matplotlib
plt.imshow(image)
plt.axis('off')  # Hide axes for a cleaner display
plt.show()

# Use the model to describe the image
llm_with_image_context = image_llm.bind(images=[base64_image])
llm_with_image_context.invoke("Describe the image")

response = requests.post(url, json=payload)
